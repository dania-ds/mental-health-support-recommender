# Mental Health Support Recommender (NLP Project)

This project is an **awareness-based mental health support system** built using **Natural Language Processing (NLP)** techniques.

This system does NOT diagnose or prescribe medication.



## Problem Statement
Users often describe mental health concerns in informal language (e.g., stress, tension).
This system maps user input to relevant mental health conditions and provides:
- Similar conditions
- Treatment categories (simplified, awareness only)
- Common side effects
